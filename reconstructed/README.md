# Horizon Deutsch Training Institute - Website Reconstruction

A professional reconstruction of the Horizon Deutsch Training Institute website, maintaining the original style while using modern web development best practices.

## Project Overview

This is a complete website reconstruction for Horizon Deutsch Training Institute, a German language training institution in Kenya. The website provides information about German language programs (A1-B2 levels), career pathways, virtual learning options, and admissions.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern CSS Framework**: Clean, maintainable CSS with CSS variables for easy customization
- **Interactive Elements**: 
  - Mobile navigation menu
  - Tab-based program level selection
  - Contact form with validation
  - Smooth scroll animations
- **SEO Optimized**: Proper meta tags and semantic HTML structure
- **Fast Loading**: Minimal dependencies and optimized code

## Pages

1. **Home** (`index.html`) - Hero section, program overview, features, and call-to-action
2. **About Us** (`about.html`) - Company vision, mission, teaching philosophy, and benefits
3. **Programs** (`programs.html`) - Detailed program information with interactive tabs for A1-B2 levels
4. **Career Pathways** (`career-pathways.html`) - Career opportunities in Germany and study options
5. **Virtual Learning** (`virtual-learning.html`) - Online learning features and how it works
6. **Admissions** (`admissions.html`) - Enrollment process, steps, and requirements
7. **Contact Us** (`contact.html`) - Contact information, map, and contact form
8. **Privacy Policy** (`privacy-policy.html`) - Privacy policy information
9. **Terms of Service** (`terms-of-service.html`) - Terms and conditions
10. **Cookie Policy** (`cookie-policy.html`) - Cookie usage information

1. Open `index.html` in a web browser to view the website

## Customization

### Changing Colors
Edit the CSS variables in `css/styles.css` under the `:root` selector:

```css
:root {
    --primary-color: #1a1a2e;
    --secondary-color: #16213e;
    --accent-color: #e94560;
    /* ... */
}
```

### Adding New Pages
1. Create a new HTML file following the existing template structure
2. Add the navigation link to all existing pages
3. Add any specific CSS to `css/styles.css` if needed

### Modifying Content
All content is in the HTML files. Simply edit the text, images, or links as needed.
